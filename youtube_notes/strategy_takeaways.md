## Rules To Add

1. Trade less, filter more.
   - Do not save/act on low quality setups.
   - Prefer A/B quality only.

2. Risk first.
   - Every plan must show real USD risk.
   - If min lot makes risk higher than target, warn clearly.

3. Journal everything.
   - Save setup context, not just entry/SL/TP.
   - Required ML fields: quality, spread, session, ATR, RSI, EMA gap, result.

4. Avoid forcing trades.
   - `NO TRADE` is a valid output.
   - Low momentum, weak trend, high spread = no signal.

5. Improve by review cycle.
   - Track TP/SL result.
   - Later add MFE/MAE to learn if entry/SL/TP is reasonable.
